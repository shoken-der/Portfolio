# Portfolio Website

This is my personal portfolio website, featuring my projects, skills, and certifications.

## Demo
The site is live on Vercel: [https://shokender.vercel.app](https://shokender.vercel.app)

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Vercel (for deployment)

## Local Development
To run this project locally, you can use any static file server. For example:
```bash
npx serve .
```
